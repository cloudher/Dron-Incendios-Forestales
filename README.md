# Dron-Incendios-Forestales
## Descripción del Proyecto

Este proyecto académico de Ingeniería Mecatrónica tiene como objetivo desarrollar una propuesta tecnológica para mejorar la detección, monitoreo y respuesta ante incendios forestales mediante el uso de un vehículo aéreo no tripulado (dron).

La solución propuesta integra sensores para la recolección de información ambiental, sistemas de monitoreo aéreo y mecanismos de apoyo a la extinción mediante el uso de agentes químicos especializados. El proyecto busca contribuir a la reducción del tiempo de respuesta ante emergencias y minimizar el impacto ambiental y económico ocasionado por los incendios forestales.

El repositorio reúne la documentación, diseños, simulaciones y desarrollos realizados durante la ejecución del proyecto.
